Esse projeto foi feito com o foco de aprender a mexer com Typescript, React+Vite e Docker.

Se trata de uma aplicação WEB que puxa informações sobre servidores do MMORPG Tibia pelo Backend, através da API TibiaData, e retorna para o Frontend.
A página principal renderiza a lista dos servidores em cards, contendo informações gerais sobre eles, ao clicar no card de um servidor você abre um painel mostrando algumas informações específicas desse servidor.
